7.1 Ejercicio | Complete los siguientes enunciados:

  • a) Todas las secuencias de comandos pueden escribirse en términos de tres tipos de estructuras de control: ___secuencia___, ___selección___ y ___repeticion___.

  • b) La instrucción de selección doble ___if...else___ se utiliza para ejecutar una acción cuando una condición es verdadera, y otra acción cuando esa condición es falsa.

  • c) Al proceso de repartir un conjunto de instrucciones un número especifico de veces se le llama repetición ___controlada por contador___.

  • d) Cuando no se sabe de antemano cuántas veces se repetirá un conjunto de instrucciones, se puede usar un valor ___centinela___ (o un ___de señal___, ___de bandera___ o ___de prueba___) para terminar la repetición.

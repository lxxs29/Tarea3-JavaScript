6.13 Ejercicio | Dada y = ax 3 + 7 . ¿cuáles de las siguientes instrucciones de JavaScript son correctas para esta ecuación?

  • a) y = a * x * x * x + 7;

    • >> Correcta.

  • b) y = a * x * x * (x + 7);

    • >> Incorrecta.

  • c) y = (a * x) * x * (x + 7);

    • >> Incorrecta.

  • d) y = (a * x) * x * x + 7;

    • >> Incorrecta.

  • e) y = a * (x * x * x) + 7;

    • >> Correcta.

  • f) y = a * x * (x * x + 7);

    • >> Incorrecta.

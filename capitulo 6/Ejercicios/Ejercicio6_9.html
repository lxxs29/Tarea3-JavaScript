6.9 Ejercicio | Indique si cada uno de los siguientes enunciados es verdader o falso. Si es falso, explique por qué.

  • a) Los operadores de JavaScript se evaluán de izquierda a derecha.

    • >> Falso, La precedencia en JS se evalua como en las matematicas, la multiplicación y la división se resuelven primero, despues la suma y la resta (la suma se evalua de izquierda a derecha ).

  • b) Los siguientes nombres de variables válidos: _guion_bajo_, m928134, t5, j7, sus_ventajas$, el_total_de_su_$cuenta, a, b$, c, z, z2.

    • >> Verdadero .

  • c) Una expresión aritmética válida de JavaScript sin paréntesis se evalúa de izquierda a derecha.

    • >> Falso, se evalua de derecha a izquierda.

  • d) Los siguientes nombres de variables inválidas: 3g,87, 67h2, h22, 2h.

    • >> Falso, JavaScript no permite la declaracion de variables con el primer caracter como un número.

6.11 Ejercicio | ¿Qué aparece en el diálogo de alerta cuando se ejecuta cada una de las siguientes instrucciones de JavaScript? Asuma que x = 2 & y = 3.

  • a) window.alert("x = " + x);

    • >> x = x

  • b) window.alert("El valor de x + x es " + (x + x));

    • >> El valor de x + x es (2x)

  • c) window.alert("x = ");

    • >> x =

  • d) window.alert( (x + y) + "=" + (y + x) );

    • >> (x + y) = (y + x)

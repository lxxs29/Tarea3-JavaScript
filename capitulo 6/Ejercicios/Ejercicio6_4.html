6.4 Ejercicio | Ideniifique y corrija en cada una de las siguientes instrucciones.

a)

if (c < 7); window.alert("c es menor que 7");

b)

if (c => 7 ) window.alert("c es igual o mayor que 7");

Corrección:

a)

if (c < 7) window.alert("c es menor que 7");

Cuando solo se usa una línea de código despues de una condición es necesario indicar que la instrucción continua, para ello en el caso de solo ser una línea siguiente, se puede dejar el fin de la condición son ; o sin {}.

b)

if (c >= 7 ){ window.alert("c es igual o mayor que 7"); }

El operador para comparar en este caso debe ser >= no =>.

6.12 Ejercicio | ¿Cuál de las siguientes instrucciones de JavaScript contiene variables cuyos valores se modifican?.

  • a) p = i + j + k + 7;

    • >> Variable con valor modificado.

  • b) window.alert("variables cuyos valores se destruyen");

    • >> No es una variable.

  • c) window.alert(" a = 5");

    • >> No es una variable.

  • d) cadenaVal = window.prompt("Escriba una cadena: ");

    • >> Variable con valor modificado.

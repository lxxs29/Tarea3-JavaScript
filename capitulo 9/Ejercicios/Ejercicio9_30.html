9.30 Ejercicio | ¿Qué hace la siguiente función?

// El parámetro b debe ser entero
// positivo para evitar recursividad infinita
function misterio( a, b )
{
 if ( b == 1 )
  return a;
 else
  return a + misterio(a, b - 1);
}

Hace la suma de a que se hace b cantidad de veces.

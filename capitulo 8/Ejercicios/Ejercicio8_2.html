8.2 Ejercicio | Escriba una instracción o un conjunto de instrucciones en JavaScript para realizar cada una de las siguientes tareas.

  • a) Sumar los impares entre 1 y 99. Utilice una instracción for. Suponga que se han declarado las variables suma y cuenta.

  • b) Calcular el valor de 2.5 elevado a la potencia de 3. Use el método pow.

  • c) Imprimir los enteros del 1 al 20 mediante el uso de un ciclo while y la variable contador x. Imprimar solo cinco entero por línea.

  • d) Repita el ejercicio 8.2 (c), pero ahora use una instrucción for.+

8.1 Ejercicio | Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué

  • a) El caso default es requerido en la instrucción de selección swtich.

    • >> Falso, default es opcional.

  • b) La instracción break es requerida en el último caso de una instracción de selección swtich.

    • >> Falso, solo se usa para salir del swtich.

  • c) La expresión (x > y && a < b) es verdadera x > y es verdadera, o si a < b es verdadera.

    • >> Falso, Ambas expresiones realcionales deben ser verdaderas para que toda la expresión sea verdadera cuando se utilice el operador &&.

  • d) Una expresión que contiene el operador || es verdadera si uno o ambos de sus operadores son verdaderos.

    • >> Verdadero.
